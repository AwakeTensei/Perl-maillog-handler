Test project for searching strings and sorting mail logs from maillog file with Perl, MySQL and HTML.

Short documentation:

DB_Table_Create.txt - create your tables in MySQL database using code from this file.

DB_Loader.pl - loading strings from maillog file into database.

DB_Searcher.pl - search strings by email address from your database. This script also contains HTML page for interactions with other scripts using interface buttons.

DB_Cleaner.pl - clean database tables.
